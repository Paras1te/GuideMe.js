Introduction

GuideMe.js is a pure javascript module for creating guides/tutorials for websites/pages with easily modifiable operations and appearance of modals. No jQuery or any other library required!

Simple Example

var guide = new GM.Guide();
guide.addStep({
   modal:
   {
      relativeTo: document.getElementById("element1"),
      title: "Modal Title",
      content: "Simple text."
   }
});
guide.start();

First we start with initializing new instance of Guide:

var guide = new GM.Guide();

Then we add our first step with modal:

guide.addStep({
   modal:
   {
      relativeTo: document.getElementById("element1"),
      title: "Modal Title",
      content: "Simple text."
   }
});

Lastly we call our guide to start:

guide.start();

That's it! This simple example shows how to create simple guide with default funcionality.

Functions have mostly default values defined, which you can leave and set the ones you want to. For more info about default values and available parameters check the documentation below.
